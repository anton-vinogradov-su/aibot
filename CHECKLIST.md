## Что дальше?

1. Настроить .env файл с вашими ключами
2. Запустить проект: `docker-compose up --build`
3. Инициализировать БД: `make init-db` (или вручную через API)
4. Проверить работу через /docs
5. Мониторить логи: `make logs`
